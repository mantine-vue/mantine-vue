# Support

This guide will help you understand how to get help, report bugs and connect with the community.
Keep in mind that Mantine Vue is maintained by a single developer, all Mantine Vue projects
are open-source and free for everyone.

## Get help

If you have any questions, need help with Mantine Vue, want to request a feature,
or just want to chat with other developers, you can:

- Ask questions using [GitHub Discussions](https://github.com/mantine-vue/mantine-vue/discussions)
- Browse the [Mantine React documentation](https://mantine.dev/) for shared concepts (theming, styles API, PostCSS preset)

## Report an issue

We use GitHub issues to track bugs. You can find a list of current open issues on
[this page](https://github.com/mantine-vue/mantine-vue/issues).

If you found a bug and want to report it, please follow these steps:

1. Search for similar issues on GitHub, maybe someone has already reported it.
2. Make sure that you are using the latest version of Mantine Vue, the issue might have been already fixed.
3. If you did not find similar issues, [create a new one](https://github.com/mantine-vue/mantine-vue/issues/new/choose),
   make sure to provide as much information as possible, including a
   [minimal reproduction example](https://stackoverflow.com/help/minimal-reproducible-example).
4. Wait for the issue to be reviewed and fixed.

## Support Mantine Vue

All contributions to the project are welcome and appreciated. There are many ways to support the project:

- Share your feedback in [GitHub Discussions](https://github.com/mantine-vue/mantine-vue/discussions) –
  we are always happy to hear your thoughts on how to improve Mantine Vue.
- Help others on [GitHub Discussions](https://github.com/mantine-vue/mantine-vue/discussions).
- Give us a code review. You are welcome to explore the
  [source code](https://github.com/mantine-vue/mantine-vue) and provide your feedback.
- Star the project on [GitHub](https://github.com/mantine-vue/mantine-vue). It is a small thing that
  helps us grow and get more people interested in the project.
- [Contribute](/contribute) to the Mantine Vue codebase.

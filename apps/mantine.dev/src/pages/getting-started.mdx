# Getting started

## Get started with a template

The easiest way to get started is to use one of the official templates.
All templates include the required dependencies and pre-configured settings.
Some templates also include additional features like [Vitest](https://vitest.dev/),
[Storybook](https://storybook.js.org/) and Oxlint.

Templates include only `@mantine-vue/core` and `@mantine-vue/hooks` packages.
If you need additional `@mantine-vue/*` packages, follow the installation instructions
of the package you want to use.

To get started with a template, open it on GitHub and click the "Use this template"
button. In order to use this feature you need to be logged in to your GitHub account.

Templates list:

- [mantine-vue-vite-template](https://github.com/mantine-vue/mantine-vue-vite-template) – Vite + Vue 3 with Vitest, Storybook and Oxlint
- [mantine-vue-vite-min-template](https://github.com/mantine-vue/mantine-vue-vite-min-template) – minimal Vite + Vue 3 setup
- [mantine-vue-nuxt-template](https://github.com/mantine-vue/mantine-vue-nuxt-template) – Nuxt with SSR, Vitest, Storybook and Oxlint

## Get started without a framework

<InstallScript packages="@mantine-vue/core @mantine-vue/hooks" />

Install PostCSS plugins and [postcss-preset-mantine](https://mantine.dev/styles/postcss-preset):

<InstallScript packages="postcss postcss-preset-mantine postcss-simple-vars" dev />

Create a `postcss.config.cjs` file at the root of your application with the following content:

```js
module.exports = {
  plugins: {
    'postcss-preset-mantine': {},
    'postcss-simple-vars': {
      variables: {
        'mantine-breakpoint-xs': '36em',
        'mantine-breakpoint-sm': '48em',
        'mantine-breakpoint-md': '62em',
        'mantine-breakpoint-lg': '75em',
        'mantine-breakpoint-xl': '88em',
      },
    },
  },
}
```

Add styles imports and wrap your application with [MantineProvider](https://mantine.dev/theming/mantine-provider/):

```ts
// src/main.ts
import { createApp, h } from 'vue'
import { MantineProvider, type MantineThemeOverride } from '@mantine-vue/core'
import App from './App.vue'

// core styles are required for all packages
import '@mantine-vue/core/styles.css'

const theme: MantineThemeOverride = {
  /** Put your mantine theme override here */
}

createApp({
  render: () => h(MantineProvider, { theme, defaultColorScheme: 'auto' }, () => h(App)),
}).mount('#app')
```

If your application has server side rendering, set an initial `data-mantine-color-scheme`
attribute on the `<html />` element to avoid a hydration flash. See the
[Usage with Nuxt](/guides/nuxt) guide for a complete SSR example.

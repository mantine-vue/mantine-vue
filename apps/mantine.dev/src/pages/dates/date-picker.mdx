## Allow deselect

Set `allowDeselect` to allow users to deselect the currently selected date by clicking on it.
`allowDeselect` is disregarded when the `type` prop is `range` or `multiple`. When a date is
deselected, `onChange` is called with `null`.

<Demo data={DatePickerDemos.deselect} />

## Presets

Use the `presets` prop to add custom date presets. Presets are displayed next to the calendar:

<Demo data={DatePickerDemos.presets} />

## Default date

Use the `defaultDate` prop to set the date value that will be used to determine which year should be displayed initially.
For example, to display February 2015, set `defaultDate="2015-02-01"`. If the value is not specified,
then `defaultDate` uses the current date. Note that if you set the `date` prop, then the `defaultDate` value will be ignored.

<Demo data={DatePickerDemos.defaultDate} />

## Hide outside dates

Set the `hideOutsideDates` prop to remove all dates that do not belong to the current month:

<Demo data={DatePickerDemos.hideOutsideDates} />

## Display week numbers

Set the `withWeekNumbers` prop to display week numbers:

<Demo data={DatePickerDemos.withWeekNumbers} />

## First day of week

Set the `firstDayOfWeek` prop to configure the first day of the week. The prop accepts a number from 0 to 6,
where 0 is Sunday and 6 is Saturday. The default value is 1 – Monday. You can also configure this option
for all components with [DatesProvider](/dates/getting-started).

<Demo data={DatePickerDemos.firstDayOfWeek} />

## Exclude dates

To disable specific dates, use the `excludeDate` prop.
It accepts a function that takes a date as an argument and returns a boolean value – if `true` is returned, the date will be disabled.
Example of disabling all dates that are not Fridays:

<Demo data={DatePickerDemos.excludeDate} />

## Additional features

`DatePicker` also supports the following [Calendar](/dates/calendar) props via prop fallthrough:
`renderDay`, `getDayProps`, `getYearControlProps`, `getMonthControlProps`, `weekendDays`, `hideWeekdays`,
`monthLabelFormat`, `yearLabelFormat`, `decadeLabelFormat`, `monthsListFormat`, `yearsListFormat`,
`highlightToday`, `withCellSpacing`, `defaultLevel`, `maxLevel`, `minLevel`. Read through the
[Calendar](/dates/calendar) documentation to learn more about them.

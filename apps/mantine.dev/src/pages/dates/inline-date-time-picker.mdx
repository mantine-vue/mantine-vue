## Usage

`InlineDateTimePicker` supports the same props as [DateTimePicker](/dates/date-time-picker).

> Note: `InlineDateTimePicker` currently renders the same dropdown-based UI as
> `DateTimePicker` rather than an always-open inline layout (calendar + time picker rendered
> directly, without a popover). This is a documented simplification.

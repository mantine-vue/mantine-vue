# Testing with Jest

This guide will help you set up [Jest](https://jestjs.io/) and
[Vue Testing Library](https://testing-library.com/docs/vue-testing-library/intro) for your project.

> **Vitest is recommended**
>
> The Vue ecosystem – including the official Vite and Nuxt tooling – uses [Vitest](/guides/vitest) for
> unit testing. We recommend Vitest for all new Mantine Vue projects. This guide is provided for teams
> that already use Jest and want to keep it.

## Installation

Install Jest, Vue Testing Library and the Vue transform:

<InstallScript
  dev
  packages="jest jest-environment-jsdom @testing-library/jest-dom @testing-library/vue @testing-library/user-event @vue/vue3-jest @babel/core babel-jest"
/>

## Configuration

Create a `jest.config.cjs` file in your project root:

```js
module.exports = {
  testEnvironment: 'jsdom',
  setupFilesAfterEnv: ['<rootDir>/jest.setup.cjs'],
  transform: {
    '^.+\\.vue$': '@vue/vue3-jest',
    '^.+\\.(t|j)sx?$': 'babel-jest',
  },
  moduleFileExtensions: ['vue', 'ts', 'tsx', 'js', 'jsx', 'json'],
  moduleNameMapper: {
    '\\.css$': '<rootDir>/test-utils/style-mock.cjs',
  },
}
```

Create a `babel.config.cjs` file so Babel can transform TypeScript and Vue sources:

```js
module.exports = {
  presets: [['@babel/preset-env', { targets: { node: 'current' } }], '@babel/preset-typescript'],
}
```

## Mock Web APIs

Most Mantine Vue components depend on browser APIs like `window.matchMedia` or `ResizeObserver`.
These APIs aren't available in the `jest-environment-jsdom` environment and you'll need to mock them.

Create a `jest.setup.cjs` file in your project root and add the following code to it:

```js
require('@testing-library/jest-dom')

const { getComputedStyle } = window
window.getComputedStyle = (elt) => getComputedStyle(elt)
window.HTMLElement.prototype.scrollIntoView = () => {}

Object.defineProperty(window, 'matchMedia', {
  writable: true,
  value: jest.fn().mockImplementation((query) => ({
    matches: false,
    media: query,
    onchange: null,
    addListener: jest.fn(),
    removeListener: jest.fn(),
    addEventListener: jest.fn(),
    removeEventListener: jest.fn(),
    dispatchEvent: jest.fn(),
  })),
})

class ResizeObserver {
  observe() {}
  unobserve() {}
  disconnect() {}
}

window.ResizeObserver = ResizeObserver
```

## Custom render

All Mantine Vue components require [MantineProvider](https://mantine.dev/theming/mantine-provider/) to be
present in the component tree. Create a custom render function that wraps the component under test:

```ts
// ./test-utils/render.ts
import { MantineProvider } from '@mantine-vue/core'
import { render as testingLibraryRender } from '@testing-library/vue'
import { h, type Component } from 'vue'
import { theme } from '../src/theme'

export function render(ui: Component) {
  return testingLibraryRender(MantineProvider, {
    props: { theme, env: 'test' },
    slots: { default: () => h(ui) },
  })
}
```

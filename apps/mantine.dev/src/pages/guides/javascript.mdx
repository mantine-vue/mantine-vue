# Usage with JavaScript

## Is it possible to use Mantine Vue with JavaScript?

Yes, it's possible to use all `@mantine-vue/*` packages (as well as all other npm packages) with
JavaScript. `@mantine-vue/*` packages are written in TypeScript and ship type definitions, so you'll
get some benefits of TypeScript (like autocompletion in your IDE) even when using them with JavaScript.

In single-file components you can simply omit the `lang="ts"` attribute on the `<script setup>` block:

```vue
<script setup>
import { Button } from '@mantine-vue/core'
</script>

<template>
  <Button>Click me</Button>
</template>
```

## Should Mantine Vue be used with JavaScript?

We recommend using Mantine Vue with TypeScript. It doesn't require deep knowledge of TypeScript
and will make your code more robust and easier to maintain. For example, you'll get type errors
when you pass invalid props to components. TypeScript will also help you during migration to new
versions of Mantine Vue.

If you're not familiar with TypeScript yet, using Mantine Vue with TypeScript will be a great
opportunity to learn it. You can use any of the [templates](/getting-started) to get started – all of
them include TypeScript support out of the box.

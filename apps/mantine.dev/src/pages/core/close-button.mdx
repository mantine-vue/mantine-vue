## Usage

`CloseButton` renders a button with an `X` icon inside. It is used in other Mantine components like [Drawer](/core/drawer) or [Modal](/core/modal).

<Demo data={CloseButtonDemos.usage} />

## Change icon

You can change the icon by passing any Vue node to the `icon` prop.
It is useful when `CloseButton` is used as a part of other components,
for example in [Drawer](/core/drawer) or [Modal](/core/modal).
Note that if you use the `icon` prop, the `iconSize` prop is ignored –
you will have to set the icon size manually.

## Accessibility

To make the `CloseButton` accessible for screen readers, set the `aria-label` prop or
use the [VisuallyHidden](/core/visually-hidden) component:

```vue
<script setup lang="ts">
import { CloseButton, VisuallyHidden } from '@mantine-vue/core'
</script>

<template>
  <!-- Option 1: aria-label -->
  <CloseButton aria-label="Close modal" />

  <!-- Option 2: VisuallyHidden child -->
  <CloseButton>
    <VisuallyHidden>Close modal</VisuallyHidden>
  </CloseButton>
</template>
```

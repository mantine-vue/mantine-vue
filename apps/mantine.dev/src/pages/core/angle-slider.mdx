## Usage

Use the `AngleSlider` component to pick an angle value between 0 and 360:

<Demo data={AngleSliderDemos.usage} />

## Controlled

The `AngleSlider` value is a number between 0 and 360:

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { AngleSlider } from '@mantine-vue/core'

const value = ref(180)
</script>

<template>
  <AngleSlider :value="value" @change="(v) => (value = v)" />
</template>
```

## AngleSlider with uncontrolled forms

`AngleSlider` can be used with uncontrolled forms. Set the `name` attribute to include the slider
value in `FormData` on form submission. Use `defaultValue` to set the initial value:

```vue
<script setup lang="ts">
import { AngleSlider } from '@mantine-vue/core'

function onSubmit(event: Event) {
  event.preventDefault()
  const formData = new FormData(event.currentTarget as HTMLFormElement)
  console.log('Angle value:', formData.get('angle'))
}
</script>

<template>
  <form @submit="onSubmit">
    <AngleSlider name="angle" :default-value="120" />
    <button type="submit">Submit</button>
  </form>
</template>
```

## Marks

Set the `marks` prop to display marks on the slider. Each mark is an object with a required
`value` (0–360) and an optional `label`. To restrict selection to marks only, set `restrictToMarks`:

<Demo data={AngleSliderDemos.marks} />

## onChangeEnd

The `onChangeEnd` callback fires when the user stops dragging or changes the value with the keyboard.

## Accessibility

Set `aria-label` to make the slider accessible to screen readers:

```vue
<template>
  <AngleSlider aria-label="Gradient angle" />
</template>
```

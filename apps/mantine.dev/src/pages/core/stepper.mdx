## Allow step select

## With custom icons

Replace the step icon by setting the `icon` prop on `Stepper.Step`.
To change the completed check icon, set `completedIcon` on the `Stepper` component.
Pass a render function `() => h(Icon, props)` as the icon value:

## Get step ref

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { Stepper } from '@mantine-vue/core'

const stepperRef = ref<HTMLDivElement | null>(null)
const firstStepRef = ref<HTMLButtonElement | null>(null)
</script>

<template>
  <Stepper ref="stepperRef" :active="0">
    <Stepper.Step ref="firstStepRef" label="First step" />
    <Stepper.Step label="Second step" />
  </Stepper>
</template>
```

## Wrap Stepper.Step

`Stepper` relies on `Stepper.Step` order. Wrapping `Stepper.Step` in another component is not supported — create a separate component for the step content instead:

```vue
<template>
  <Stepper :active="1">
    <Stepper.Step label="Regular step">First step</Stepper.Step>
    <!-- Do not wrap Stepper.Step — children will not render -->
    <Stepper.Step label="Step with custom content">
      <MyStepContent />
    </Stepper.Step>
  </Stepper>
</template>
```

## Accessibility

`Stepper.Step` renders a `<button>` element. Set `aria-label` or `title` when no `label` or `description` is provided:

```vue
<template>
  <Stepper :active="0">
    <!-- Not ok – no label for screen readers -->
    <Stepper.Step />

    <!-- Ok – label and description -->
    <Stepper.Step label="Step 1" description="Create an account" />

    <!-- Ok – aria-label -->
    <Stepper.Step aria-label="Create an account" />
  </Stepper>
</template>
```

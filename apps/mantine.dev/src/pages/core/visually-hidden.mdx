## Usage

`VisuallyHidden` is a utility component that hides content visually but leaves it available to screen readers.

For example, it can be used with [ActionIcon](/core/action-icon) component:

```vue
<script setup lang="ts">
import { PhHeart } from '@phosphor-icons/vue'
import { ActionIcon, VisuallyHidden } from '@mantine-vue/core'
</script>

<template>
  <ActionIcon>
    <PhHeart />
    <VisuallyHidden>Like post</VisuallyHidden>
  </ActionIcon>
</template>
```

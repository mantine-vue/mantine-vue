## Underline

Use the `underline` prop to configure the `text-decoration` property. It accepts the following values:

- `always` - link is always underlined
- `hover` - link is underlined on hover
- `never` - link is never underlined
- `not-hover` - link is underlined when not hovered

<Demo data={AnchorDemos.decoration} />

You can also configure the `underline` prop for all `Anchor` components with default props:

```vue
<script setup lang="ts">
import { Anchor, createTheme, MantineProvider } from '@mantine-vue/core'

const theme = createTheme({
  components: {
    Anchor: Anchor.extend({
      defaultProps: {
        underline: 'always',
      },
    }),
  },
})
</script>

<template>
  <MantineProvider :theme="theme">
    <!-- Your app here -->
  </MantineProvider>
</template>
```

## order prop

The `order` prop sets the heading level for the `Accordion.Control` root element.
According to WAI-ARIA recommendations, it is required to use `h2`-`h6` heading levels
to fit correctly within the outline of the page.

## Change chevron

Use the `chevron` prop to change the chevron icon. To remove it, set `:chevron="null"`.

To customize chevron styles, use the Styles API with the `data-rotate` attribute
(set when the item is open, unless `disableChevronRotation` is set):

## Default opened items

For `multiple={false}`, set `defaultValue` as a string. For `multiple={true}`, set it as an array:

```vue
<template>
  <!-- Single: second item opens by default -->
  <Accordion defaultValue="item-2">
    <Accordion.Item value="item-1"><!-- ... --></Accordion.Item>
    <Accordion.Item value="item-2"><!-- ... --></Accordion.Item>
  </Accordion>

  <!-- Multiple: both items open by default -->
  <Accordion multiple :defaultValue="['item-1', 'item-2']">
    <Accordion.Item value="item-1"><!-- ... --></Accordion.Item>
    <Accordion.Item value="item-2"><!-- ... --></Accordion.Item>
  </Accordion>
</template>
```

## Control opened state

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { Accordion } from '@mantine-vue/core'

const value = ref<string | null>(null)
</script>

<template>
  <Accordion :value="value" @change="value = $event">
    <Accordion.Item value="item-1"><!-- ... --></Accordion.Item>
    <Accordion.Item value="item-2"><!-- ... --></Accordion.Item>
  </Accordion>
</template>
```

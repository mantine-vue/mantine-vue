## Data type

`OverflowList` accepts any array via the `data` prop. Pass `renderItem` and `renderOverflow`
as function props to control how visible items and the overflow indicator are rendered:

```vue
<script setup lang="ts">
import { OverflowList } from '@mantine-vue/core'

const data = [{ value: '1', label: 'Item 1' }]
</script>

<template>
  <OverflowList
    :data="data"
    :renderItem="(item) => h('div', { key: item.value }, item.label)"
    :renderOverflow="(items) => h('div', {}, \`+\${items.length} more\`)"
  />
</template>
```

## Max rows

Use `maxRows` to limit visible rows count. By default, 1 row is visible.

## Max visible items

Use `maxVisibleItems` to limit visible items count. By default, there is no limit.

<Demo data={OverflowListDemos.maxVisibleItems} />

## Collapse from

Use `collapseFrom` prop to control from which direction items are collapsed
when they overflow. By default, items are collapsed from the end. Set
`collapseFrom="start"` to collapse items from the beginning – this is useful
for breadcrumb-like patterns where the last items should remain visible.

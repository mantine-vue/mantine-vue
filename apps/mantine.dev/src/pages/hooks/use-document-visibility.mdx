## Usage

The `use-document-visibility` hook returns a readonly ref with the current [document.visibilityState](https://developer.mozilla.org/en-US/docs/Web/API/Document/visibilityState)
– it allows you to detect if the current tab is active:

// DocumentVisibilityState is 'visible' | 'hidden'

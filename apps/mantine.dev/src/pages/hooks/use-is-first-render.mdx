## Usage

The `useIsFirstRender` hook returns a readonly ref that is `true` before the component has updated
for the first time, and `false` afterward. Note that since it relies on Vue's `onUpdated` lifecycle
hook, the value flips to `false` only after the component actually re-renders (not on a timer or on mount).

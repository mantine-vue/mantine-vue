## Usage

The `use-favicon` hook appends a `<link />` element to the `head` with the given favicon in an isomorphic effect.
The hook is not triggered during server-side rendering.

Call the hook with a favicon URL (supported formats: `.ico`, `.png`, `.svg`, and `.gif`), passed as a plain string,
a `Ref`, or a getter function (`MaybeRefOrGetter<string>`). The favicon is updated every time the value changes and
is not an empty string or `null`.

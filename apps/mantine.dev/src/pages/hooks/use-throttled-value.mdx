## Usage

The `useThrottledValue` hook accepts a `MaybeRefOrGetter` value and a wait time in milliseconds.
It returns a readonly ref with a throttled value that cannot change more than once every `wait` milliseconds.

## Definition

```ts
function useThrottledValue<T>(value: MaybeRefOrGetter<T>, wait: number): Readonly<Ref<T>>
```

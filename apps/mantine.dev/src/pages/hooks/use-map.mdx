## Usage

The `useMap` hook returns a reactive [Map](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Map)
object built with Vue's `reactive` – calling `set`, `clear`, or `delete` on it (or on entries read out of it)
updates any component that reads from the map and triggers a rerender.

## Definition

```tsx
function useMap<T, V>(initialState?: [T, V][]): Map<T, V>
```

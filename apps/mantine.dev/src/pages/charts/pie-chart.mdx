## Usage

`PieChart` displays proportional data. Each data item requires `name` and `value` and can define a Mantine color.

<Demo data={ChartsDemos.pieChart} />

## Labels

Enable `withLabels` and `withLabelsLine` to display segment labels. Use `labelsType="percent"` for percentage values.

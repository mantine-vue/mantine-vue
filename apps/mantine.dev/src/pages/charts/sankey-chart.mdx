## Usage

`SankeyChart` displays weighted flow between nodes. Links reference node names or indexes through `source` and `target`.

<Demo data={ChartsDemos.sankeyChart} />

## Data structure

Pass `nodes` and `links`. Every link requires `source`, `target`, and numeric `value`; nodes require unique names.

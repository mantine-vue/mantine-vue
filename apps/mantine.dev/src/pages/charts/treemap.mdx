## Usage

`Treemap` displays hierarchical data as nested rectangles whose areas correspond to values.

<Demo data={ChartsDemos.treemap} />

## Nested data

Each node accepts `name`, optional `value`, `color`, and recursive `children`. Leaf values determine the visible area.

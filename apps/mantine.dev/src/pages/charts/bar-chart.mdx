## Usage

`BarChart` supports grouped, stacked, percent, and waterfall datasets and both horizontal and vertical orientations.

<Demo data={ChartsDemos.barChart} />

## Value labels

Set `withBarValueLabel` to display values and use `maxBarWidth`, `minBarSize`, and `fillOpacity` to tune bar geometry.

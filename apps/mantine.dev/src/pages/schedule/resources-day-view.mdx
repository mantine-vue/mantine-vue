## Usage

ResourcesDayView displays resources as rows and time slots as columns. Assign an event to a row with its `resourceId` property.

<Demo data={ResourcesDayViewDemos.usage} />

## All-day events

Foreground all-day events are pinned to the top of their resource row. Background all-day events tint the row.

<Demo data={ResourcesDayViewDemos.allDayEvents} />

## Time range and intervals

Use `startTime`, `endTime` and `intervalMinutes` to control the visible time range and granularity.

## Custom resource label

Use `renderResourceLabel` with a VNode, or the `resourceLabel` named slot. Named slots are also available for the header, corner, time labels, time slots, groups, events, all-day and background events, more-events control, and current-time indicator.

## Keyboard navigation

When a time slot is focused, arrow keys move between times and resource rows. Static mode removes all slots and events from keyboard interaction.

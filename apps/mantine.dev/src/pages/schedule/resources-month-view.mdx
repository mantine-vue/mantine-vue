## Usage

ResourcesMonthView displays resources as rows and every day of the selected month as columns. Events use `resourceId` to select their row.

<Demo data={ResourcesMonthViewDemos.usage} />

## Static mode

<Demo data={ResourcesMonthViewDemos.staticMode} />

## Drag and drop

Events can move between days and resources. `onEventDrop` receives the target `resourceId`.

<Demo data={ResourcesMonthViewDemos.dragDrop} />

## Event form

Use `onDayClick`, `withDragSlotSelect` and `onSlotDragEnd` to create events for a resource.

<Demo data={ResourcesMonthViewDemos.eventForm} />

# @mantine-vue/schedule

[![npm](https://img.shields.io/npm/dm/@mantine-vue/schedule)](https://www.npmjs.com/package/@mantine-vue/schedule)

Schedule and calendar components for Mantine Vue.

[View documentation](https://mantine-vue.dev/schedule/getting-started)

## Installation

```bash
# With yarn
yarn add @mantine-vue/schedule @mantine-vue/core @mantine-vue/hooks @mantine-vue/dates

# With npm
npm install @mantine-vue/schedule @mantine-vue/core @mantine-vue/hooks @mantine-vue/dates
```
